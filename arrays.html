What are arrays?

An array is a new type of variable.

It is a container where we can store lists of other variables.

What are they used for?

Any time you want to store a collection of similiar things, think about using an array.

Create an empty array


fruits = []
=> []

(Handy hint - it looks like a box for your variables!)


fruits = Array.new
=> []

Create an array with some items


fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]

(Some programming languages need you to specify how many elements your array will have, but Ruby doesn't.)

Accessing arrays

Let's look at the array we created earlier.


fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]
						

fruits[0]
=> "apple"
fruits[1]
=> "pear"

Very important - arrays are "zero-indexed" - that means they start at 0 instead of 1!

So the first array item can be found from:


fruits[0]
=> "apple"

What will this display?


fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]

fruits[100]
						

=> nil

We can also access elements from the end of the array:


fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]

fruits[-1]
=> "banana"

There are some handy utility methods for getting the first and last items:


fruits.first
=> "apple"

fruits.first(2)
=> ["apple", "pear"]

fruits.last
=> "banana"

fruits.last(2)
=> ["tangerine", "banana"]

Adding items to an array

You can assign by index:


fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]

fruits[4] = "peach"
=> "peach"

What will this do?


fruits[100] = "plum"
=> "plum"

(Check the fruits variable afterwards!)

We can also add items to the start or end of the array:


fruits.push("plum")
=> ["apple", "pear", "tangerine", "banana", "plum"]
						

fruits << "plum"
=> ["apple", "pear", "tangerine", "banana", "plum"]
						

fruits.unshift("plum")
=> ["plum", "apple", "pear", "tangerine", "banana"]

Removing items from an array

If you know what you want to remove, you can call the fruits.delete method:


fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]

fruits.delete("apple")
=> ["pear", "tangerine", "banana"]

You can remove the first or last element from the array:


fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]

fruits.pop
=> "banana"
						

fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]

fruits.shift
=> "apple"

Array elements can be any type of object, literal value, or variable.

Unlike some other languages, Ruby lets us use a mixture within a single array:


fruits_and_numbers = ["apple", 1, "pear", 2]
=> ["apple", 1, "pear", 2]

We can even put arrays inside arrays!


numbers = [1, 2, 3, 4, [5, 6, 7]]

What will numbers.count give us?

What will numbers.pop give us?

Try these now!

Using operators, we can:

Add arrays together


[1, 2, 3] + [3, 4, 5]
=> [1, 2, 3, 3, 4, 5]

Remove elements from an array


[1, 2, 3] - [3, 4, 5]
=> [1, 2]

Check which elements exist in both arrays


[1, 2, 3] & [3, 4, 5]
=> [3]

Return elements that are in either array


[1, 2, 3] | [3, 4, 5]
=> [1, 2, 3, 4, 5]

More handy methods

These should give you a flavour of some of the things you can do with arrays.

Check the documentation for more detail.

(Press the down arrow)

Count elements in an array:


fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]

fruits.size
=> 4

Reverse elements in an array:


fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]

fruits.reverse
=> ["banana", "tangerine", "pear", "apple"]

Cycle through, or rotate through an array:


fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]

fruits.rotate
=> ["pear", "tangerine", "banana", "apple"]

Shuffle array elements randomly:


fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]

fruits.shuffle
=> ["pear", "banana", "apple", "tangerine"]

Return a random array element:


fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]

fruits.sample
=> "pear"

fruits.sample(2)
=> ["tangerine", "apple"]

We can ask arrays about their content:


fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]

fruits.any? {|fruit| fruit.length < 5}
=> true

fruits.all? {|fruit| fruit.length < 10}
=> true

fruits.none? {|fruit| fruit.length < 3}
=> true

fruits.include? "apple"
=> true

fruits.max
=> "tangerine"

fruits.min
=> "apple"

We can manipulate arrays to change their structure:


["apple", "pear"].concat( ["plum"] )
=> ["apple", "pear", "plum"]

["apple", ["tangerine", "banana"]].flatten
=> ["apple", "tangerine", "banana"]

["apple", "pear", nil, nil].compact
=> ["apple", "pear"]

["apple", "apple", "pear"].uniq 
=> ["apple", "pear"]

We can transform arrays into other objects (for example strings):


["apple", "pear", "tangerine", "banana"].join(" / ")
=> "apple / pear / tangerine / banana"

Important!

When we call a method on an array, we get a new array back. If we want to change the original array, we can use a bang method.


fruits = ["apple", "pear", "tangerine", "banana"]
=> ["apple", "pear", "tangerine", "banana"]

fruits.sort!
=> ["apple", "banana", "pear", "tangerine"]

This lets us permanently change an array.
